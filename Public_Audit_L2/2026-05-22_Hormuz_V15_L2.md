### HEI计算公式(修正版) | HEI Formula (Revised)

```
HEI = 0.4 × F + 0.3 × M + 0.3 × (1-D)

5月22日计算:
F = 0.85 (PGSA正式运转 + 三级通行体系 + 海底光缆收费提议 + 通行量极低 + 联合国决议对峙)
M = 0.82 (布伦特约104-105; SPR单周-992万桶; 全球库存创纪录消耗)
(1-D) = 1-0.35 = 0.65 (铀浓缩对立 + 美方内部信号极度矛盾 + 外交轨道质量贬值)

HEI = 0.4×0.85 + 0.3×0.82 + 0.3×0.65 = 0.340 + 0.246 + 0.195 = 0.781 ≈ 0.78 ± 0.08
```

**修正前后对比:** 0.67 → 0.78 (+0.11)。更精准契合当前"美军登船、光缆收费、安理会决议对撞、SPR倾库"的极高摩擦现实。
