## 三、可复现验证路径（Replicable Verification Protocols）

### 3.1 弗迪产能覆盖比率（月度复现模板）

**步骤**：
1. 下载工信部第XXX批公告，统计"弗迪电池有限公司"配套比亚迪纯电动乘用车条目数 → N
2. 取公告中各车型电池系统能量（kWh），按条目数加权平均 → Avg_kWh
3. 月度装车量估算 = N × Avg_kWh / 1000 → X GWh
4. 取比亚迪当月新能源乘用车销量 → Y 万辆
5. 理论电池需求 = Y × 55 kWh（保守估计） → Z GWh
6. **覆盖比率 = X / Z × 100%**

**判定规则**：
- <50%：产能严重不足，维持🔴极高风险评级
- 50%-70%：爬坡中，方向性判断维持
- >70%：证伪当前判定，瓶颈解除

**首次验证时间**：2026年6月15日

### 3.2 碳酸锂Q3方向验证（双周复现模板）

**探针1**：广期所碳酸锂仓单量（连续4周环比增长≥10%→供给宽松信号）
**探针2**：海关总署HS 2601.12项下锂矿进口量（7月数据环比+100%→方向确认）
**探针3**：上海钢联/百川盈孚现货均价（跌破12万或突破25万→趋势确认）

**决策规则**：
- 探针1+探针2同时满足 → 偏向高盛6.5万/吨情景，Beta路径概率上调至50%+
- 探针1未触发+现货站稳20万 → 维持区间震荡判断

### 3.3 欧盟绿证认可风险（持续追踪模板）

**探针1**：EUR-Lex《新电池法》碳足迹核算授权法案更新状态
**探针2**：CATARC与欧盟JRC对接进展公告
**探针3**：TÜV/SGS对中国电芯企业碳足迹认证排期时长（>9个月且延长→窗口关闭信号）

**触发条件**：2026年Q3前EUR-Lex无明确认可条款 → 风险从方向性升级为确认性，启动对欧出口暂停预案。
