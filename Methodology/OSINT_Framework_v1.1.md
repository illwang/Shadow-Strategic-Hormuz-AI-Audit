## 第一章 信息架构层

### 1.1 三层物理隔离结构

| 层级 | 定义 | 准入门槛 | 使用边界 |
|:---|:---|:---|:---|
| **T1 确认层** | 已发生、可独立复现的事实 | 须有≥1条🟢或🟡级免费验证路径；多源公开核实，置信度≥85% | 禁止付费数据库独占验证 |
| **T2 推断层** | 基于T1的合理推演 | 方向性判断（60-85%）或推测性观察（≤60%）；允许付费研报摘要辅助 | 须标注来源与局限 |
| **T3 叙事层** | 无法验证的传闻、市场情绪、政治放风 | 禁止进入正文 | 仅入附录"未验证信号观察区"，标注来源与升级条件 |

### 1.2 OSINT可验证性原则

- T1确认性判断必须附带至少一条免费验证路径（URL / 公开数据库 / API / 官方公告）
- 仅能通过🔴付费数据库验证的数据点，不得进入T1，仅可作为T2或T3
- **分析师不持有、不阅读、不引用任何机密/付费报告原文**
- 接受外部机构的"验证结论"（如"偏差约15%"），但不接受机密数据原文
- 所有对外发布内容仍**100%基于OSINT**

### 1.3 信源-内容交叉评级矩阵（规则零）

```
                内容可信度
                   A         B         C
            ┌─────────┬─────────┬─────────┐
         S  │   S1    │   S2    │   S3    │  ← 官方/一手数据
信源等级    ├─────────┼─────────┼─────────┤
         N  │   N1    │   N2    │   N3    │  ← 主流通讯社/智库
            ├─────────┼─────────┼─────────┤
         C  │   C1    │   C2    │   C3    │  ← 自媒体/传闻
            └─────────┴─────────┴─────────┘
```

> **准入门槛**：低于C3的信号禁入AIPI计算，必须达B2或以上。

### 1.4 论证强度分级体系

| 等级 | 置信度 | 使用规范 |
|:---|:---|:---|
| 确认性判断 | ≥85% | 多源公开核实，可进入T1 |
| 方向性判断 | 60%-85% | 基于T1的合理推演，可进入正文 |
| 推测性观察 | ≤60% | 逻辑推演，须明确标注不确定性 |
| 分析性推断 | 逻辑推演 | 情景化表述，**禁止赋概率百分比** |

**三不原则**：
1. 不为定性事件赋概率
2. 不为机制套机械公式
3. 不隐瞒"无法量化"的区域

### 1.5 六条永久性校验规则

**规则一：半导体新架构量产周期校验**  
新芯片架构量产自动套用**18个月缓冲期**。缓冲期内，任何"即将量产"声明须降级为T2或T3，禁止进入T1。

**规则二：材料替代弹性三问**  
任何材料管制声明须回答：
1. 功能替代是否存在（化学/电学 vs 机械/结构）？
2. 认证周期多长？
3. 库存缓冲能撑多久？
> 三问中任一答案为"否/未知/不足"，则该材料瓶颈进入T1。

**规则三：地缘政治声明的物理/法律可行性双过滤**
- **物理层**：技术、物流、基础设施是否支持执行？
- **法律层**：国际法、国内法、合同框架是否赋予合法性？
> 任一不通过则降级为姿态（T3），不进入T1/T2。

**规则四：财务收入质量三拆**
任何财务数据进入T1前须拆解：
1. 现金/信用 —— 应收账款占比是否异常？
2. 关联方/第三方 —— 收入是否来自关联交易？
3. 消耗/承诺 —— 收入是已交付还是仅合同签署？

**规则五：滞后数据声明**
涉及官方滞后数据时，若使用第三方估算，必须标注：  
> "待官方数据验证，预计发布时间为[日期]"

**规则六：信号污染隔离区**
以下五类信号禁止直接进入核心判断，须经清洗后方可按规则零至五进入后续流程：
1. 市场情绪噪声
2. 政治放风
3. 企业公关声明
4. 社交媒体放大效应
5. 资本叙事操纵

## 第二章 战略压力评估层

### 2.1 全谱系战略压力图（L0-L6）

| 层级 | 定义 | 典型要素 |
|:---|:---|:---|
| **L0** 地理集中风险 | 产能地理集中度、单点故障 | 自然灾害、地缘冲突、旱情/地震 |
| **L1** 核心芯片与先进封装 | 逻辑芯片、存储芯片、封装产能 | GPU/ASIC、HBM/DDR、CoWoS/Chiplet |
| **L2** 高速互联 | 光模块、网络拓扑 | EML/VCSEL、硅光子、光纤/电缆 |
| **L3** 机电基础设施 | 电力与物理安全 | 变压器、柴发、SMR/微反应堆、BESS |
| **L4** 半导体材料 | 关键原材料 | 氦气、光刻胶、氟化氢、硅晶圆、稀土 |
| **L4.5** 被动元件 | 全链通胀蔓延信号层 | MLCC、电阻、电感 |
| **L5** 环境合规 | 水资源与法规约束 | 液冷渗透率、环评、欧盟AI法案 |
| **L6** 金融层 | 资本可持续性 | CAPEX、战争险费率、估值泡沫、FCF |

### 2.2 层间耦合状态分析规范

每条活跃耦合线须回答三个问题：
1. **触发条件是什么？**（量化阈值或具体事件）
2. **传导机制是什么？**（物理/金融/政策/信息）
3. **当前状态是真实压力变化，还是评分者校准调整，还是方法论修正？**

**耦合状态分类**：
- `休眠` —— 当前无显著耦合
- `活跃` —— 存在直接传导路径
- `内部活跃` —— 同一层级内部子模块联动
- `警戒` —— 跨层风险可能系统性传导

### 2.3 AIPI战略压力等级（AIPI-Level）

**三维加权模型**：

```
AIPI = 0.40 × S + 0.35 × D + 0.25 × G
```

**S（供给刚性）子维度权重**：

| 子维度 | 权重 | 说明 |
|:---|:---|:---|
| 核心即时供给（HBM/先进制程） | 0.35 | 当前最紧缺环节 |
| 先进封装产能 | 0.20 | CoWoS/Chiplet物理上限 |
| 光互联/高速传输器件 | 0.15 | EML/硅光子瓶颈 |
| 能源基础设施 | 0.10 | 变压器/柴发/电力并网 |
| 半导体材料 | 0.15 | 氦气/光刻胶/氟化氢 |
| 被动元件 | 0.05 | MLCC等通胀蔓延信号 |

**D（需求验证）子维度权重**：

| 子维度 | 权重 | 说明 |
|:---|:---|:---|
| 闭源模型商业化质量 | 0.35 | ARR/FCF/收入结构 |
| 开源模型效率提升 | 0.25 | MoE/异构计算/HBM绕过 |
| 云商/头部客户CAPEX可持续性 | 0.25 | CAPEX增速 vs 云收入增速 |
| 新兴需求极 | 0.15 | 具身智能、边缘计算等 |

**G（地缘裂度）子维度权重**：

| 子维度 | 权重 | 说明 |
|:---|:---|:---|
| 中美科技博弈 | 0.30 | 管制/审批/技术封锁 |
| 材料出口管制对抗 | 0.25 | 光刻胶/氟化氢/稀土相互反制 |
| 关键产区劳工/地缘化 | 0.20 | 罢工、地震、旱情 |
| 物理脆弱性 | 0.15 | 台海/台积电集中风险 |
| 航运/能源地缘裂度 | 0.10 | 海峡封锁、战争险费率 |

**评分标准**：每个子维度0-1分，1=完全安全/宽松，0=完全断裂/极端紧张

**综合得分区间**：

| 等级 | 区间 | 状态定义 |
|:---|:---|:---|
| AIPI-A | 0.80-1.00 | 战略宽松状态 |
| AIPI-B | 0.70-0.79 | 温和承压 |
| AIPI-C | 0.60-0.69 | 结构性紧张 |
| AIPI-D | 0.50-0.59 | 战略脆弱状态 |
| AIPI-E | <0.50 | 极端失稳 |

### 2.4 AIPI变动审计表（强制字段）

| 字段 | 内容要求 |
|:---|:---|
| 上一期读数 | 精确到小数点后两位 |
| 本期读数 | 精确到小数点后两位 |
| 变动幅度 | 正负数值 |
| 主要驱动维度 | S/D/G中主导变化的一个 |
| 变动归因 | 具体事件或数据变化 |
| 次要驱动维度 | 第二个显著变化的维度 |
| 次要归因 | 具体事件或数据变化 |
| 变动性质 | `真实压力变化` / `评分者校准` / `方法论修正` |

### 2.5 敏感性推演与并发情景

必须包含四类推演：
1. **单变量推演**：每个关键事件的上行情景与下行情景
2. **双变量并发推演**：两个关键事件同时发生的净效应
3. **极端并发推演**：三个及以上变量同时冲击的系统性效应
4. **效率革命/技术颠覆独立监测**：建立触发阈值与行动框架

> **局限性声明（强制）**：主观判断、±0.08的底层不确定性、OSINT边界、极端事件失效。当任一L0或L1核心子维度达到"极端失稳"条件时，直接激活S级补充报告。

|:---|:---|:---|

## 第四章 应对策略层

### 4.1 地缘经济弹性模型

**弹性等级量化标准（十分制加权）**：

```
综合得分 = 0.50 × 供应稳定性 + 0.25 × 替代速度 + 0.25 × 成本可承受度
```

| 等级 | 区间 |
|:---|:---|
| ★★★★★ | ≥9.0 |
| ★★★★ | 7.5-8.9 |
| ★★★ | 6.0-7.4 |
| ★★ | 4.5-5.9 |
| ★ | <4.5 |

### 4.2 弹性策略编号体系

**格式**：`EL-[层级代码]-[序号]`

**层级代码对照表**：

| 代码 | 含义 |
|:---|:---|
| L0 | 地理集中风险 |
| L1 | 核心芯片 |
| L2 | 高速互联 |
| L3 | 机电基础设施 |
| L4 | 半导体材料 |
| L5 | 环境合规 |
| L6 | 金融层 |
| HBM | HBM专项 |
| PKG | 先进封装 |
| H200 | 双源算力 |
| OPT | 光互联 |
| POW | 能源电力 |
| MAT | 材料 |
| ENV | 环境/液冷 |
| FIN | 金融对冲 |
| CLS | 闭源模型API |

**每条策略须包含**：
- 弹性等级（★级）
- 成本（高/中/低）
- 回报周期（立即/短期/中期/长期）
- 执行时限（P0/P1/P2，或具体日期）
- 最坏后悔值（1=完全正确，5=重大不可逆损失）
- 衰减条件（什么情况下后悔值会下降/归零）
- 复核日期

### 4.3 后悔值评估标准

针对每条弹性策略，评估其在**四种终局情景**下的后悔程度：

1. **软着陆**：AI ARR质量可靠，持续超越成本，FCF转正
2. **效率革命**：技术突破将核心需求降至当前1/3以下
3. **资本断流**：连续两季CAPEX负增长，运营现金流无法覆盖
4. **地缘断裂**：核心供给全面禁运

> 以最坏情景得分标注当前后悔值。

### 4.4 决策半衰期表

| 时间窗口 | 监测事件 | 受影响策略 | 窗口状态 | 半衰期定义 | 剩余时间 | 后悔值 |
|:---|:---|:---|:---|:---|:---|:---|
| 1-4周 | ... | EL-... | 正在关闭 | 截止日期 | X天 | ★★★★★ |
| 1-3月 | ... | EL-... | 收窄中 | 审批结束 | X周 | ★★★★☆ |
| 6-12月 | ... | EL-... | 开放但紧迫 | 数据发布 | X月 | ★★★☆☆ |

### 4.5 证伪条件与反向推演

**证伪条件**（任一触发即推翻对应核心判断）：

| 情景 | 触发条件 | 推翻内容 |
|:---|:---|:---|
| 技术合作突破 | 签署正式协议；实质性放宽管制 | 制裁路径恶化判断 |

| 效率革命 | 实测需求降至1/3以下 | 刚性依赖；短缺逻辑 |

**反向推演**（从极端情景倒推触发条件）：

|:---|:---|:---|

| 产区立场转变 | 关键产区收紧出口 | 对华可获得性判断 |
| 产能严重中断 | 自然灾害/冲突致停产超两周 | 全球供给基本假设 |
| 基础设施遭打击 | 关键设施遭系统性打击 | 投资能力假设 |

## 第五章 监测与迭代层

### 5.1 代理指标监测面板设计规范

**设计原则**：
- 纯数据陈列，不指向具体资产操作，**不构成交易建议**
- 使用高频公开数据弥补低频官方数据空白
- 付费数据源必须提供免费替代监测方案

**面板字段**：

| 监测领域 | 代理指标 | 免费路径 | 更新频率 | 当前读数 | 方向 | 风险信号 |
|:---|:---|:---|:---|:---|:---|:---|
| ... | ... | 🟢/🟡 | 每日/周/月 | ... | ↑↓→ | 阈值触发条件 |

**三色阈值**：
- 🟢 **低风险**：历史正常区间
- 🟡 **中等风险**：偏离正常区间但未触极值
- 🔴 **高风险**：接近或超越历史极值，触发AIPI重新评估

### 5.2 叙事疲劳监测（NFI）

| 叙事 | 疲劳度评估 | 反转风险 |
|:---|:---|:---|
| "长期紧缺" | 高疲劳 | 任何新增产能或需求放缓信号都可能反转 |
| "脱钩不可逆" | 中疲劳 | 实质性合作信号可能快速瓦解叙事 |
| "技术颠覆一切" | 低疲劳 | 关键验证未兑现则叙事可能消退 |

### 5.3 外部报告加成验证机制

**操作流程**：
1. 每期OSINT报告发布后，识别**"信息缺口清单"**
2. 标注："以下判断存在公开信息缺口，建议通过商业数据验证"
3. 接收客户反馈（**仅接受"验证结论"，不接受"机密数据原文"**）
4. 将验证结论纳入下期AIPI校准，但不引用任何机密报告内容

**合规防火墙**：
- 分析师不持有、不阅读、不引用任何机密/付费报告原文
- 仅接收"验证结论"（如"与TrendForce一致"或"偏差约15%"）
- 所有对外发布内容仍**100%基于OSINT**
- 若客户要求引用机密数据，引导客户自行在内部使用

|:---|:---|:---|:---|:---|
| 第N轮 | ... | ... | ... | 已修正/待修正 |

**框架迭代触发条件**：
- **方法论缺陷**：任何审计轮次发现系统性方法论问题 → 立即迭代
- **数据验证失败**：T1层数据被证伪 → 当期内修正，下期更新方法论
- **新工具需求**：分析新产业链时发现现有工具不足 → 季度评估后迭代
